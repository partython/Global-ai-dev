DEPLOYMENT QUICK START
================================================================================

1. Prerequisites:
   - AWS Account with appropriate permissions
   - Terraform >= 1.5.0
   - kubectl >= 1.29
   - AWS CLI v2
   - S3 bucket for state
   - DynamoDB table for locking

2. Deploy Infrastructure (20-30 minutes):
   cd terraform/
   terraform init
   terraform plan -var-file=environments/production.tfvars
   terraform apply

3. Configure Kubernetes (5 minutes):
   aws eks update-kubeconfig --name priya-global-production
   
4. Deploy Kubernetes Resources (10 minutes):
   kubectl apply -f k8s/namespace.yaml
   kubectl apply -f k8s/configmap.yaml
   kubectl apply -f k8s/services.yaml
   kubectl apply -f k8s/ingress.yaml
   kubectl apply -f k8s/hpa.yaml
   kubectl apply -f k8s/network-policy.yaml

5. Deploy Microservices (30-60 minutes):
   kubectl apply -f deployments/*.yaml

6. Verify & Test (15 minutes):
   kubectl get pods -A
   curl https://api.priya-global.com/health

Total Time: ~2-3 hours for full deployment
