## Key Features Across All Documents

### Compliance Coverage
- GDPR (EU data protection)
- UK GDPR & Data Protection Act 2018
- CCPA (California privacy)
- CASL (Canadian anti-spam)
- PIPEDA (Canadian privacy)
- LGPD (Brazilian privacy)
- CAN-SPAM (US anti-spam)
- GDPR Article 28 (processor obligations)

### Data Protection
- AES-256 encryption at rest
- TLS 1.3 encryption in transit
- SOC2 Type II certification target
- RBAC and MFA implementation
- Encrypted backups
- 90-day post-termination deletion
- Data breach notification (24 hours)

### Service Providers (Sub-processors)
- **Infrastructure:** AWS (data storage, hosting)
- **Payments:** Stripe (PCI-DSS compliant)
- **AI:** OpenAI, Anthropic (with opt-out)
- **Communications:** Twilio, SendGrid
- **Analytics:** Google Analytics, Sentry
- **Monitoring:** Third-party integrations

### Key Policies
- 99.9% uptime SLA with service credits
- 14-day money-back guarantee (annual plans)
- 30 days notice for material changes
- Service credits only remedy for downtime
- Binding arbitration for disputes
- Indemnification obligations
- Customer data ownership maintained

### Rate Limits
- API: 1,000 requests/minute (standard)
- Messages: 10,000 per hour per channel
- File uploads: 100 MB per file, 10 GB per day
- Concurrent connections: 100
- Enforcement: 429 status, progressive rate limiting

### Enforcement Actions
- **Minor violations:** Warning notices (30-day cure period)
- **Moderate violations:** 7-30 day suspension
- **Serious violations:** Permanent termination
  - Phishing/fraud
  - Malware distribution
  - CSAM/child exploitation
  - DDoS attacks
  - IP theft

## Usage Guidance

### For Marketing/Sales
- Reference Privacy Policy for customer trust
- Share Terms of Service during onboarding
- Highlight data ownership and security measures

### For Enterprise Customers
- Provide Data Processing Agreement with signed agreement
- Discuss data residency options (US, EU, UK)
- Review sub-processor list and data flow

### For Compliance/Legal
- DPA satisfies GDPR Article 28 requirements
- Audit rights available per DPA Section 10
- Sub-processor changes notified 30 days in advance
- Breach notification procedures documented

### For Support/Operations
- Reference AUP for enforcement decisions
- Use rate limit sections for capacity management
- Follow data deletion procedures (90 days post-termination)
- Use contact information in each document for inquiries
