Getting Started

1. Authentication

Register a user or login to get JWT tokens. Use the access token in Authorization header for all requests.

2. Multi-tenancy

All requests are tenant-scoped. X-Tenant-ID is automatically injected by the gateway.

3. Rate Limiting

Plans have different rate limits. Check X-RateLimit-* headers in responses.

4. Channels

Connect WhatsApp, Email, SMS, Voice, Social, and other channels to reach customers.

5. AI Integration

Use the AI Engine to provide intelligent responses with knowledge base support.

6. Analytics

Track conversations, messages, and customer satisfaction with real-time dashboards.
